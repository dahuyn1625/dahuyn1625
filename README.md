## Featured Projects

### 1. Fruit Detection with EfficientDet
**Mục tiêu**: Xây dựng mô hình object detection thời gian thực phục vụ nông nghiệp thông minh, nhận diện trái cây chính xác trong môi trường thực tế.  

**Tech Stack**: PyTorch, EfficientDet, OpenCV, Albumentations, ONNX + TensorRT.  

**Kết quả đạt được**:  
• Đạt **93.7% mAP@0.5** trên dataset tùy chỉnh  
• Tối ưu latency cho edge deployment  
• Pipeline hoàn chỉnh từ data labeling đến REST API inference  

**Hướng mở rộng**: Tích hợp multi-modal (drone + satellite imagery) và triển khai MLOps pipeline.  

[→ Xem Repository](https://github.com/YOUR_GITHUB_USERNAME/fruit-detection-efficientdet)

### 2. LLM Chatbot vs RAG Chatbot (Comparative Study)
**Mục tiêu**: So sánh và chứng minh giá trị của Retrieval-Augmented Generation trong việc giảm hallucination và tăng độ chính xác cho hệ thống hỏi đáp domain-specific.  

**Tech Stack**: LangChain, Hugging Face Transformers, FAISS vector store, Llama-3 / Mistral, FastAPI.  

**Kết quả đạt được**:  
• Giảm tỷ lệ hallucination **~65%** so với vanilla LLM  
• Xây dựng production-ready asynchronous API  
• Advanced chunking & hybrid retrieval strategy  

**Hướng mở rộng**: Phát triển multi-modal RAG và agentic AI workflows.  

[→ Xem Repository](https://github.com/YOUR_GITHUB_USERNAME/rag-llm-comparison)

### 3. Time Series Forecasting Engine
**Mục tiêu**: Xây dựng framework dự báo chuỗi thời gian đa chân trời với độ chính xác cao và uncertainty quantification.  

**Tech Stack**: PyTorch, Temporal Fusion Transformer, Darts, Prophet.  

**Kết quả đạt được**:  
• Cải thiện MAE **28%** so với baseline truyền thống  
• Hỗ trợ multi-horizon forecasting & probabilistic outputs  

**Hướng mở rộng**: Áp dụng cho energy, finance và anomaly detection systems.
